WhiteCanvas

The main PHP classes:

Autoloader

The autoloader use a list of folders where your PHP classes can be found when required.

These folders are defined in _CONFIG_DIR_/json/autoloaded.json by a constant or a path.

Route

The route class use an array of uri to controllers to find the right controller for the current requested uri.

These uri/controllers pairs can be defined in _CONFIG_DIR_/json/routes.json by the uri and the full class name.

Controller

The controller's abstract class defines the run method that every controllers should at least have and a set of tools that can be relevent such as:

  • The possibility of implementing a checkToken method that will be called first when run is called.
  • The setTranslation method that will set the current language to a lang value provided by the request.
  • The getAction method that will search a method with a 'process' prefix to run instead of the default render method.
  • An alias to the translate method "_t" to ease the use of the translation class inside a controller.
  • The couples addJS fecthJS addCSS fetchCSS methods to facilitate the linking of css or js files into your templates.
  • And finally a processRenderHelper to make ajax calls possible without having to write one line in the targeted controller.
